Notation

Here you will find all the notation used throughout all subjects available on this website:
iff If and Only If
s.t. Such That
DNE Does Not Exist
$\in$ Belongs To or Is an Element Of
$\exists$ There Exists
$\forall$ For All
$\mathbb{N}$ The Set of Natural Numbers
$\mathbb{Z}$ The Set of Integers
$\mathbb{Q}$ The Set of Rational Numbers
$\mathbb{R}$ The Set of Real Numbers
$\mathbb{R}_{> 0}$ The Set of Positive Real Numbers
$\mathbb{R}_{\geq 0}$ The Set of Nonnegative Real Numbers
$f:A \rightarrow B$ The Mapping $f$ from $A$ to $B$
$\subset$ Proper Subset
$\subseteq$ Subset
$A \setminus B$ Set Difference: All Objects in $A$ and not $B$
