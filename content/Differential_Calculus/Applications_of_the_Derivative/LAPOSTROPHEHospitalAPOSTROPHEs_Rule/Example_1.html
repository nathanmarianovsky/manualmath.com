Problem
Evaluate the following using L'Hospital's Rule:
$\eqalign{ \lim_{x \rightarrow \infty} \frac{x^2 - 9x + 4}{3x^2 + 7x + 8} }$
Solution
The initial limit will approach:
$\eqalign{ \lim_{x \rightarrow \infty} \frac{x^2 - 9x + 4}{3x^2 + 7x + 8} = \frac{\infty}{\infty} }$
This provides reasoning to use L'Hospital's Rule:
$\eqalign{ \lim_{x \rightarrow \infty} \frac{x^2 - 9x + 4}{3x^2 + 7x + 8} &= \lim_{x \rightarrow \infty} \frac{2x - 9}{6x + 7} \\ &= \frac{\infty}{\infty} }$
Since the result is still an indeterminate form, we use the rule once more:
$\eqalign{ \lim_{x \rightarrow \infty} \frac{2x - 9}{6x + 7} &= \lim_{x \rightarrow \infty} \frac{2}{6} \\ &= \frac{1}{3} }$
