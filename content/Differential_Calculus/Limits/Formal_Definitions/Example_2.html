Problem
Prove:
$\eqalign{ \lim_{x \rightarrow a} c = c \hspace{.3cm} \text{where} \hspace{.3cm} c \in \mathbb{R} }$
Solution
By the formal definition a limit:
$\eqalign{ \lim_{x \rightarrow a} f(x) = L }$
will exist if $\forall \epsilon > 0, \exists \delta > 0$ s.t.:
$\eqalign{ |f(x) - L| < \epsilon \hspace{.3cm} \text{whenever} \hspace{.3cm} |x - a| < \delta }$
In this scenario the function of interest is $f(x) = c$ giving:
$\eqalign{ |f(x) - L| &< \epsilon \\ |c - c| &< \epsilon \\ |0| &< \epsilon \\ 0 &< \epsilon }$
which holds true because it is initially assumed that $\epsilon > 0$. Now it does not matter what the value of $\epsilon$ is, and as for $\delta$ any positive value can be chosen because no matter the $\delta$ the initial condition is already satisfied.
