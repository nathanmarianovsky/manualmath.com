Problem
Prove that:
$\eqalign{ \lim_{x \rightarrow 4} \sqrt{x + 5} = 3 }$
by using the $(\delta, \epsilon)$ definition of the limit.
Solution
By the formal definition a limit:
$\eqalign{ \lim_{x \rightarrow a} f(x) = L }$
will exist if $\forall \epsilon > 0, \exists \delta > 0$ s.t.:
$\eqalign{ |f(x) - L| < \epsilon \hspace{.3cm} \text{whenever} \hspace{.3cm} |x - a| < \delta }$
So we want to find a $\delta$ in terms of $\epsilon$ s.t.:
$\eqalign{ |x - 4| < \delta }$
Using the other inequality we arrive at:
$\eqalign{ |f(x) - L| &< \epsilon \\ |\sqrt{x + 5} - 3| &< \epsilon \\ |\sqrt{x + 5} - 3||\sqrt{x + 5} + 3| &< |\sqrt{x + 5} + 3|\epsilon \\ |(x + 5) - 9| &< |\sqrt{x + 5} + 3|\epsilon \\ |x - 4| &< |\sqrt{x + 5} + 3|\epsilon }$
The only question remaining now is how do we get rid of the factor on the right side. You can choose an arbitrary value such as $\delta = 1$ which provides $|x - 4| < 1 \implies 3 < x < 5$. Using this we can deduce:
$\eqalign{ |x - 4| &< |\sqrt{x + 5} + 3|\epsilon \\ &< (\sqrt{8} + 3)\epsilon }$
which minimizes the right hand-side by letting $x = 8$. So now we have the issue that there are two possibilities for $\delta$. With two choices, we should choose the smaller one to be more accurate. So as long as we choose $\delta = \min\Big\{ 1, (\sqrt{8} + 3)\epsilon \Big\}$, the inequalities will hold true, thus making the limit true.
