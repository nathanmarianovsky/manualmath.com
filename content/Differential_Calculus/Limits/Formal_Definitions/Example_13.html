Problem
You may assume that $g(y)$ is continuous and:
$\eqalign{ \lim_{x \rightarrow a} f(x) &= L_1 \\ \lim_{y \rightarrow L_1} g(y) &= L_2 \\ }$
Use this to prove that:
$\eqalign{ \lim_{x \rightarrow a} g(f(x)) = L_2 }$
by using the $(\delta, \epsilon)$ definition of the limit.
Solution
The given information can be reformatted. We know $\forall \epsilon,\epsilon_1 > 0, \exists \delta, \delta_1 > 0$ s.t.:
$\eqalign{ |g(y) - L_2| < \epsilon \hspace{.3cm} &\text{whenever} \hspace{.3cm} |y - L_1| < \delta \\ |f(x) - L_1| < \epsilon_1 \hspace{.3cm} &\text{whenever} \hspace{.3cm} |x - a| < \delta_1 }$
Using the first line and setting $\epsilon_1 = \delta$ we are allowed to rewrite the second line as:
$\eqalign{ |f(x) - L_1| < \delta \hspace{.3cm} &\text{whenever} \hspace{.3cm} |x - a| < \delta_1 }$
So now if $|x - a| < \delta_1$, then:
$\eqalign{ |g(f(x)) - L_2| < \epsilon \hspace{.3cm} &\text{whenever} \hspace{.3cm} |f(x) - L_1| < \delta }$
which proves:
$\eqalign{ \lim_{x \rightarrow a} g(f(x)) = L_2 }$
and is the same as saying:
$\eqalign{ \lim_{x \rightarrow a} g(f(x)) = g\Big( \lim_{x \rightarrow a} f(x) \Big) }$
