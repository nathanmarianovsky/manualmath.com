Problem
Prove that:
$\eqalign{ \lim_{x \rightarrow 2} (x^2 + x - 2) = 4 }$
by using the $(\delta, \epsilon)$ definition of the limit.
Solution
By the formal definition a limit:
$\eqalign{ \lim_{x \rightarrow a} f(x) = L }$
will exist if $\forall \epsilon > 0, \exists \delta > 0$ s.t.:
$\eqalign{ |f(x) - L| < \epsilon \hspace{.3cm} \text{whenever} \hspace{.3cm} |x - a| < \delta }$
So we want to find a $\delta$ in terms of $\epsilon$ s.t.:
$\eqalign{ |x - 2| < \delta }$
Using the other inequality we arrive at:
$\eqalign{ |f(x) - L| &< \epsilon \\ |(x^2 + x - 2) - 4| &< \epsilon \\ |x^2 + x - 6| &< \epsilon \\ |(x + 3)(x - 2)| &< \epsilon \\ |x + 3||x - 2| &< \epsilon \\ |x - 2| &< \frac{\epsilon}{|x + 3|} }$
The only question remaining now is how do we get rid of the denominator. You can choose an arbitrary value such as $\delta = 1$ which provides:
$\eqalign{ |x - 2| < 1 \implies 1 < x < 3 }$
Using this we can deduce:
$\eqalign{ |x - 2| &< \frac{\epsilon}{|x + 3|} \\ &< \frac{\epsilon}{6} }$
which minimizes the right hand-side by letting $x = 3$. So now we have the issue that there are two possibilities for $\delta$. With two choices, we should choose the smaller one to be more accurate. So as long as we choose $\delta = \min\Big\{ 1, \frac{\epsilon}{6} \Big\}$ the inequalities will hold true, thus making the limit true.
