Introduction
Having seen limits and how they are determined, you should now wonder if there exist properties that could simplify calculations for us. Fortunately, there are and below you will a list of the properties exhibited by all limits.
Linearity
Limits are linear operators, thus:
$\eqalign{ \lim_{x \rightarrow a} (\alpha f(x) + \beta g(x)) &= \alpha \lim_{x \rightarrow a} f(x) + \beta \lim_{x \rightarrow a} g(x) \\ \text{where} \hspace{.3cm} &\alpha, \beta \in \mathbb{R} }$
iff the limit of each function exists.
Multiplication
Limits involving multiplication can be split:
$\eqalign{ \lim_{x \rightarrow a} (f(x) \cdot g(x)) = \lim_{x \rightarrow a} f(x) \cdot \lim_{x \rightarrow a} g(x) }$
iff the limit of each function exists.
Division
Limits involving division can be split
$\eqalign{ \lim_{x \rightarrow a} \frac{f(x)}{g(x)} &= \Big[ \lim_{x \rightarrow a} f(x) \Big] \Big[ \lim_{x \rightarrow a} g(x) \Big]^{-1} \\ \text{where} \hspace{.3cm} g(x) &\neq 0 \hspace{.3cm} \text{and} \hspace{.3cm} \lim_{x \rightarrow a} g(x) \neq 0 }$
iff the limit of each function exists.
Constant
All limits involving just a constant do nothing:
$\eqalign{ \lim_{x \rightarrow a} c = c \hspace{.3cm} \text{where} \hspace{.3cm} c \in \mathbb{R} }$
Power Function
All limits involving a power function are evaluated as:
$\eqalign{ \lim_{x \rightarrow a} x^n = a^n \hspace{.3cm} \text{where} \hspace{.3cm} n \in \mathbb{R} }$
Power
All limits involving a function raised to a power are evaluated as:
$\eqalign{ \lim_{x \rightarrow a} (f(x))^n = \Big[ \lim_{x \rightarrow a} f(x) \Big]^n \hspace{.3cm} \text{where} \hspace{.3cm} n \in \mathbb{R} }$
