Problem
Given that:
$\eqalign{ f(x) = \begin{cases} x^2 + 2x & x < a \\ -1 & x \geq a \end{cases} }$
Find all values of $a$ s.t. the function is continuous everywhere.
Solution
Independently, the components are already continuous. So we just have to make sure that at the splitting point the function is continuous. At $x = a$ we must have the components equate:
$\eqalign{ (a)^2 + 2(a) &= -1 \\ a^2 + 2a + 1 &= 0 \\ (a + 1)^2 &= 0 \\ a &= -1 }$
