Problem
Given that:
$\eqalign{ f(x) = \begin{cases} x & x \leq 3 \\ A + \frac{x}{2} & x > 3 \end{cases} }$
Find a value for $A$ s.t. the function is continuous at the point for all values of $x$.
Solution
The only part where the function may not be continuous is the splitting. So to make the function continuous at $x = 3$ we want to equate the two components to each other at the point:
$\eqalign{ (3) &= A + \frac{(3)}{2} \\ A &= \frac{3}{2} }$
