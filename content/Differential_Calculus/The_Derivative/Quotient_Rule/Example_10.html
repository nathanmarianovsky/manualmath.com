Problem
Derive the following relationship:
$\eqalign{ \frac{d}{dx} \Bigg[ \prod_{i = 1}^n f_i(x) \Bigg]^{-1} = -\sum_{i = 1}^n \frac{f_i'(x)}{f_i^2(x)} \Bigg[ \prod_{k \neq i} f_k(x) \Bigg]^{-1} }$
assuming that for all values of $i$, each $f_i(x) \neq 0$ and is differentiable at $x$.
Solution
To tackle this it might be easier if we first set the denominator to be a single function so that the derivative is right away given by:
$\eqalign{ \frac{d}{dx} \frac{1}{h(x)} = -\frac{h'(x)}{(h(x))^2} }$
when using the quotient rule. Now it is just a simple task of determining the derivative of the denominator. Fortunately we have already covered this before in Example 9 for the product rule, which provides:
$\eqalign{ &-h'(x)(h(x))^{-2} \\ &= -\Bigg[ \sum_{i = 1}^n f_i'(x) \prod_{k \neq i} f_k(x) \Bigg] \Bigg[ \prod_{i = 1}^n f_i^2(x) \Bigg]^{-1} \\ &= -\sum_{i = 1}^n \frac{f_i'(x)}{f_1^2(x)\dots f_{n - 1}^2(x)f_n^2(x)} \prod_{k \neq i} f_k(x) \\ &= -\sum_{i = 1}^n \frac{f_i'(x)}{f_i^2(x)} \Bigg[ \prod_{k \neq i} f_k(x) \Bigg]^{-1} }$
