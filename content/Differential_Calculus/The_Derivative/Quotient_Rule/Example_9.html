Problem
Derive the following relationship:
$\eqalign{ \frac{d}{dx} \Bigg[ \sum_{i = 1}^n f_i(x) \Bigg]^{-1} = -\sum_{i = 1}^n f_i'(x) \Bigg[ \sum_{j = 1}^n f_j(x) \Bigg]^{-2} }$
assuming that for all values of $i$, each $f_i(x)$ is differentiable at $x$ and the total sum does not equate to zero.
Solution
To tackle this it might be easier if we first set the denominator to be a single function so that the derivative is right away given by:
$\eqalign{ \frac{d}{dx} \frac{1}{h(x)} = -\frac{h'(x)}{(h(x))^2} }$
when using the quotient rule. Now it is just a simple task of determining the derivative of the denominator, which is a rather trivial task since the derivative is a linear operator:
$\eqalign{ -h'(x)(h(x))^{-2} &= -\Bigg[ \frac{d}{dx} \sum_{i = 1}^n f_i(x) \Bigg] \Bigg[ \sum_{j = 1}^n f_i(x) \Bigg]^{-2} \\ &= -\sum_{i = 1}^n f_i'(x) \Bigg[ \sum_{j = 1}^n f_j(x) \Bigg]^{-2} }$
