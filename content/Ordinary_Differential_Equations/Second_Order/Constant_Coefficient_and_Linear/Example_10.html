Problem
Prove that:
$\eqalign{ y(x) = C \hspace{.5cm} \text{where} \hspace{.5cm} C \in \mathbb{R} \setminus \{ 0 \} }$
is a solution to:
$\eqalign{ y'' + k_1y' + k_2y = 0 }$
iff $k_2 = 0$.
Solution
Proving iff statements requires doing both directions. First if we assume that $k_2 = 0$, then the differential equation reduces down to:
$\eqalign{ y'' + k_1y' = 0 }$
where $y(x) = C$ is without a doubt a solution. Now if we assume that $y(x) = C$ is a solution, then when plugged in the differential equation reduces down to:
$\eqalign{ k_2 \cdot C = 0 }$
We began with the given information that $y(x) \neq 0$, which tells us that $k_2 = 0$.
