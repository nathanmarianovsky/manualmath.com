Setup
Given a non-homogeneous constant coefficient linear second order equation:
$\eqalign{ y'' + ay' + by = f(x) }$
the method of undetermined coefficients provides a systematic way of guessing the non-homogeneous solution depending on the form of $f(x)$.
Typical Guesses
First determine the homogeneous solution to the differential equation and proceed to this table which lists all of the common cases and how each one is handled:
$\textbf{f(x)}$ $\textbf{y}_{nh}$
$p(x)$ $A_0 + A_1x + A_2x^2 + \dots + A_nx^n$
$ke^{\alpha x}$ $Ae^{\alpha x}$
$p(x)e^{\alpha x}$ $(A_0 + A_1x + A_2x^2 + \dots + A_nx^n)e^{\alpha x}$
$k\cos(\alpha x)$ $A\cos(\alpha x) + B\sin(\alpha x)$
$k\sin(\alpha x)$ $A\cos(\alpha x) + B\sin(\alpha x)$
$k_1\cos(\alpha x) + k_2\sin(\alpha x)$ $A\cos(\alpha x) + B\sin(\alpha x)$
$p(x)(\cos(\alpha x) + \sin(\alpha x))$ $(A_0 + \dots + A_nx^n)(\cos(\alpha x) + \sin(\alpha x))$
$(k_1\cos(\alpha x) + k_2\sin(\alpha x))e^{\beta x}$ $(A\cos(\alpha x) + B\sin(\alpha x))e^{\beta x}$
$p(x)(k_1\cos(\alpha x) + k_2\sin(\alpha x))e^{\beta x}$ $\eqalign{ \Big( \Big( \sum_{i = 0}^n A_i x^i \Big) \cos(\alpha x) + \Big( \sum_{i = 0}^n B_i x^i \Big) \sin(\alpha x) \Big)e^{\beta x} }$
where $k,k_1,k_2,\dots,k_n,\alpha,\beta \in \mathbb{R}$ represent given constants, $p(x)$ is a $n$th degree polynomial, and $A,B,C,A_0,B_0,A_1,B_1,A_2,B_2,\dots,A_n,B_n \in \mathbb{R}$ represent the coefficients that have to be determined in order to write down the non-homogeneous solution. To find the coefficients, just plug into the non-homogeneous differential equation and determine their values. If it happens that $f(x)$ is actually a sum of different cases, just use the principal of non-homogeneous linear differential equations.
Non-Homogeneous Guess is Part of Homogeneous Solution
Sometimes it may happen that the guess for the non-homogeneous solution is already part of the homogeneous solution. Fortunately, there is a fix. Take the standard guess for the non-homogeneous solution and multiply it by $x$. This idea comes from the fact that when a root is repeated in a characteristic polynomial, we form another independent solution by multiplying by $x$. To build off this, there can actually be another case. If it happens that you have for example $y_1 = x$ and $f(x) = x^3$, you would normally guess a third degree polynomial. Unfortunately, it is not that easy. Since these both share a common root, or rather eigenvalue, we still have to guess a multiple of it, so the correct guess would be a fourth degree polynomial.
