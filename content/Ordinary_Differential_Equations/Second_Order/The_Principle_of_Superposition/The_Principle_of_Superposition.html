Statement for Homogeneous Equations
Given a homogeneous linear second order differential equation in the form:
$\eqalign{ 0 = y'' + f(x)y' + g(x)y }$
and there exist two independent solutions s.t.:
$\eqalign{ 0 &= y_1'' + f(x)y_1' + g(x)y_1 \\ 0 &= y_2'' + f(x)y_2' + g(x)y_2 }$
then the general solution is defined as:
$\eqalign{ y_h(x) = c_1y_1(x) + c_2y_2(x) }$
where:
$\eqalign{ c_1,c_2 \in \mathbb{R} }$
and $h$ stands for homogeneous. To prove this just plug into the differential equation to confirm that it satisfies as a solution:
$\eqalign{ 0 &= y_h'' + f(x)y_h' + g(x)y_h \\ 0 &= (c_1y_1 + c_2y_2)'' + f(x)(c_1y_1 + c_2y_2)' + g(x)(c_1y_1 + c_2y_2) \\ 0 &= c_1(y_1'' + f(x)y_1' + g(x)y_1) + c_2(y_2'' + f(x)y_2' + g(x)y_2) \\ 0 &= c_1(0) + c_2(0) \\ 0 &= 0 }$
Statement for Non-Homogeneous Equations
Given a non-homogeneous linear second order differential equation in the form:
$\eqalign{ k_1(x) + k_2(x) + k_3(x) + \dots + k_n(x) = y'' + f(x)y' + g(x)y }$
and the solutions that satisfy:
$\eqalign{ 0 &= y_h'' + f(x)y_h' + g(x)y_h \\ k_1(x) &= j_1'' + f(x)j_1' + g(x)j_1 \\ k_2(x) &= j_2'' + f(x)j_2' + g(x)j_2 \\ k_3(x) &= j_3'' + f(x)j_3' + g(x)j_3 \\ &\vdots \\ k_n(x) &= j_n'' + f(x)j_n' + g(x)j_n }$
then the general solution to the equation is given as:
$\eqalign{ y(x) = y_h(x) + y_{nh}(x) }$
where:
$\eqalign{ y_{nh}(x) = j_1(x) + j_2(x) + j_3(x) + \dots + j_n(x) }$
To distinguish from the homogeneous solution, $nh$ stands for the non-homogeneous solution. To prove that this works just plug into the original differential equation to confirm that it satisfies as a solution:
$\eqalign{ k_1(x) + k_2(x) + k_3(x) + \dots + k_n(x) &= y'' + f(x)y' + g(x)y \\ \sum_{i = 1}^n k_i(x) &= (y_h + y_{nh})'' + f(x)(y_h + y_{nh})' + g(x)(y_h + y_{nh}) \\ \sum_{i = 1}^n k_i(x) &= (y_h'' + f(x)y_h' + g(x)y_h) + (y_{nh}'' + f(x)y_{nh}' + g(x)y_{nh}) \\ \sum_{i = 1}^n k_i(x) &= 0 + \sum_{l = 1}^n (j_l'' + f(x)j_l' + g(x)j_l) \\ \sum_{i = 1}^n k_i(x) &= \sum_{l = 1}^n k_l(x) \\ 0 &= 0 }$
Note that while the homogeneous solution will have undetermined constants, the non-homogeneous solution will never have this. The easiest way to see this is to check that if instead we took a multiple of the non-homogeneous solution and plugged in, it would not satisfy as a solution. Also note that books typically reference the non-homogeneous solution as the particular solution, but I will refrain from doing that so as to not confuse it with the particular solution that references the solution to a differential equation with initial condition(s).
