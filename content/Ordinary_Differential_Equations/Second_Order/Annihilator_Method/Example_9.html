Problem
Derive the annihilator operator for:
$\eqalign{ f(x) = ke^{\alpha x} \hspace{.5cm} \text{where} \hspace{.5cm} k,\alpha \in \mathbb{R} }$
using only the derivative operator and its corresponding eigenvalue and eigenfunction.
Solution
We begin by restating that for the derivative operator we have:
$\eqalign{ Df(x) = \alpha f(x) }$
where $f(x)$ is the eigenfunction and $\alpha$ is the associated eigenvalue. Now notice that if we bring everything to one side:
$\eqalign{ Df(x) - \alpha f(x) &= 0 \\ (D - \alpha)f(x) &= 0 }$
we get the annihilator operator for the derivative's eigenfunction. We already know that the eigenfunction happens to be exactly the function given to us, so the above annihilator is the one we want.
