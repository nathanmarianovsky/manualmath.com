Problem
Given the differential equation:
$\eqalign{ y'' + y' = 3 }$
determine the general solution using the annihilator method.
Solution
We begin by noting that the annihilator operator from the table is going to be given by:
$\eqalign{ A(D) = D }$
Now rewriting the differential equation using operators and applying the annihilator to both sides gives:
$\eqalign{ 3 &= y'' + y' \\ 3 &= (D^2 + D)y \\ D(3) &= (D)(D^2 + D)y \\ 0 &= (D)(D^2 + D)y }$
With this we are left with solving the characteristic polynomial:
$\eqalign{ 0 &= (\lambda)(\lambda^2 + \lambda) \\ 0 &= (\lambda^2)(\lambda + 1) \\ \lambda &= -1, 0 }$
where one of the roots repeats twice. Knowing this we can write down the fundamental set of solutions as:
$\eqalign{ \mathcal{S} = \{ e^{-x}, 1, x \} }$
We now form our general solution as:
$\eqalign{ y(x) = c_1e^{-x} + c_2 + c_3x }$
and plug into the differential equation to determine the constants for the non-homogeneous component:
$\eqalign{ 3 &= y'' + y' \\ 3 &= (c_1e^{-x}) + (-c_1e^{-x} + c_3) \\ c_3 &= 3 }$
With this the general solution is given by:
$\eqalign{ y(x) = c_1e^{-x} + c_2 + 3x }$
