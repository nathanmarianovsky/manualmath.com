Problem
Given the autonomous system:
$\eqalign{ \frac{dy}{dx} = \frac{ye^y - 9y}{e^y} }$
determine all the equilibrium points and graph the phase plot. For different initial points $(-2,-5)$, $(0,1)$, and $(10,\ln(9))$ determine what happens to each particular solution as $x \rightarrow \infty$.
Solution
First to obtain all the equilibrium points:
$\eqalign{ \frac{dy}{dx} &= 0 \\ \frac{ye^y - 9y}{e^y} &= 0 \\ ye^y - 9e^y &= 0 \\ y(e^y - 9) &= 0 \\ y &= 0, \ln(9) }$

Now to determine the behavior of each particular solution at very large $x$ use the phase plot:

The first point will follow the lowest region, the second point will follow the middle region, and the third point already lies on an equilibrium point and so it will stay there.
