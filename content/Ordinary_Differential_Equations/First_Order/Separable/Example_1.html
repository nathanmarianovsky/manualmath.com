Problem
Solve the following differential equation:
$\eqalign{ y' = \frac{3x^2 + 4x - 4}{2y - 4} }$
where $y(1) = 3$.
Solution
Since the function is separable:
$\eqalign{ y' &= \frac{3x^2 + 4x - 4}{2y - 4} \\ \frac{dy}{dx} &= \frac{3x^2 + 4x - 4}{2y - 4} \\ (2y - 4) dy &= (3x^2 + 4x - 4) dx \\ \int (2y - 4) dy &= \int (3x^2 + 4x - 4) dx \\ y^2 - 4y &= x^3 + 2x^2 - 4x + C \\ y^2 - 4y - x^3 - 2x^2 + 4x &= C }$
The above solution could have been put in explicit form, but that would involve using square roots, which I prefer to avoid for the time being. Now to satisfy the intial value we are given that $y(1) = 3$ which is the same as saying that $(1,3)$ is a point that must lie on the curve of the solution:
$\eqalign{ C &= y^2 - 4y - x^3 - 2x^2 + 4x \\ &= (3)^2 - 4(3) - (1)^3 - 2(1)^2 + 4(1) \\ &= -2 }$
With this the final solution can be written as:
$\eqalign{ y^2 - 4y - x^3 - 2x^2 + 4x = -2 }$
In case an implicit solution does not satisfy you, here is how the explicit solution can be found using the quadratic formula:
$\eqalign{ -2 &= y^2 - 4y - x^3 - 2x^2 + 4x \\ 0 &= (y^2) - (4y) - (x^3 + 2x^2 - 4x - 2) \\ y &= \frac{4 \pm \sqrt{16 + 4(x^3 + 2x^2 - 4x - 2)}}{2} \\ y &= 2 \pm \sqrt{x^3 + 2x^2 - 4x + 2} }$
Now a singular questions remains: which root to pick? This can only be checked if an initial value is given. In this case the point $(1,3)$ must lie on the curve:
$\eqalign{ y(1) &= 2 + \sqrt{(1)^3 + 2(1)^2 - 4(1) + 2} = 2 + \sqrt{1} = 3 \\ y(1) &= 2 - \sqrt{(1)^3 + 2(1)^2 - 4(1) + 2} = 2 - \sqrt{1} = 1 }$
Since the positive root produces the needed result, it is the correct solution in explicit form:
$\eqalign{ y = 2 + \sqrt{x^3 + 2x^2 - 4x + 2} }$
