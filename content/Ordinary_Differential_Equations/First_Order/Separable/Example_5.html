Problem
Solve the following differential equation:
$\eqalign{ \frac{dy}{dt} = 1 + t^2 + y^2 + t^2y^2 }$
where $y(0) = 1$.
Solution
Since the function is separable:
$\eqalign{ \frac{dy}{dt} &= 1 + t^2 + y^2 + t^2y^2 \\ \frac{dy}{dt} &= (1 + t^2) + (1 + t^2)y^2 \\ \frac{dy}{dt} &= (1 + y^2)(1 + t^2) \\ \frac{dy}{1 + y^2} &= (1 + t^2) dt \\ \int \frac{dy}{1 + y^2} &= \int (1 + t^2) dt \\ \arctan(y) &= t + \frac{1}{3}t^3 + C \\ \arctan(y) - t - \frac{1}{3}t^3 &= C }$
Now if we want to, finding the explicit solution can easily be achieved from the implicit solution:
$\eqalign{ \arctan(y) - t - \frac{1}{3}t^3 &= C \\ \arctan(y) &= t + \frac{1}{3}t^3 + C \\ y &= \tan\Big( t + \frac{1}{3}t^3 + C \Big) }$
To finish off the initial value has to be used to determine the value of the constant of integration:
$\eqalign{ 1 &= y(0) \\ 1 &= \tan\Big( 0 + \frac{1}{3}(0)^3 + C \Big) \\ 1 &= \tan(C) \\ C &= \frac{\pi}{4} }$
With this the final solution can be written as:
$\eqalign{ y &= \tan\Big( t + \frac{1}{3}t^3 + \frac{\pi}{4} \Big) }$
