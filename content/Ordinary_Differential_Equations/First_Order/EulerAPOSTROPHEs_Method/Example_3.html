Problem
Given the differential equation:
$\eqalign{ \frac{dy}{dx} = e^{-y} }$
and the initial condition $y(0) = 0$, determine $y(0.5)$ using a $h = 0.1$ with Euler's Method up to four decimal places and compare the result to the exact solution by finding the error.
Solution
Using Euler's Method:
$n$ $x_{n + 1} = x_n + h$ $y_{n + 1} = y_n + hf(x_n,y_n)$
0 $x_1 = x_0 + h = 0.1$ $y_1 = y_0 + hf(x_0,y_0) = 0.1000$
1 $x_2 = x_1 + h = 0.2$ $y_2 = y_1 + hf(x_1,y_1) = 0.1905$
2 $x_3 = x_2 + h = 0.3$ $y_3 = y_2 + hf(x_2,y_2) = 0.2732$
3 $x_4 = x_3 + h = 0.4$ $y_4 = y_3 + hf(x_3,y_3) = 0.3493$
4 $x_5 = x_4 + h = 0.5$ $y_5 = y_4 + hf(x_4,y_4) = 0.4198$
So according to Euler's Method $y(0.5) \approx 0.4198$. To solve for exactly separate:
$\eqalign{ \frac{dy}{dx} &= e^{-y} \\ e^y dy &= dx \\ \int e^y dy &= \int dx \\ e^y &= x + C \\ y &= \ln|x + C| }$
Now using the initial condition:
$\eqalign{ y &= \ln|x + C| \\ 0 &= \ln|(0) + C| \\ C &= 1 }$
which gives the final solution:
$\eqalign{ y = \ln|x + 1| }$
Now to determine the exact value:
$\eqalign{ y(0.5) &= \ln|(0.5) + 1| &= 0.4055 }$
and so the error is:
$\eqalign{ \Delta = \Big| \hspace{.1cm} \text{Exact} \hspace{.1cm} - \text{Approximation} \hspace{.1cm} \Big| = |0.4055 - 0.4198| = 0.0143 }$
