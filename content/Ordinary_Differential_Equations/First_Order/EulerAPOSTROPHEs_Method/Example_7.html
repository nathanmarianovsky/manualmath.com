Problem
Given the differential equation:
$\eqalign{ \frac{dy}{dx} + 2y = 2 - e^{-4x} }$
and the initial condition $y(0) = 1$, determine $y(0.5)$ using a $h = 0.1$ with Euler's Method up to four decimal places and compare the result to the exact solution by finding the error.
Solution
Using Euler's Method:
$n$ $x_{n + 1} = x_n + h$ $y_{n + 1} = y_n + hf(x_n,y_n)$
0 $x_1 = x_0 + h = 0.1$ $y_1 = y_0 + hf(x_0,y_0) = 0.9000$
1 $x_2 = x_1 + h = 0.2$ $y_2 = y_1 + hf(x_1,y_1) = 0.8530$
2 $x_3 = x_2 + h = 0.3$ $y_3 = y_2 + hf(x_2,y_2) = 0.8375$
3 $x_4 = x_3 + h = 0.4$ $y_4 = y_3 + hf(x_3,y_3) = 0.8399$
4 $x_5 = x_4 + h = 0.5$ $y_5 = y_4 + hf(x_4,y_4) = 0.8517$
So according to Euler's Method $y(0.5) \approx 0.8517$. To solve for exactly find the integrating factor and proceed:
$\eqalign{ \frac{dy}{dx} + 2y &= 2 - e^{-4x} \\ e^{2x}y' + 2e^{2x}y &= 2e^{2x} - e^{-2x} \\ \frac{d}{dx}(e^{2x}y) &= 2e^{2x} - e^{-2x} \\ d(e^{2x}y) &= (2e^{2x} - e^{-2x}) dx \\ \int d(e^{2x}y) &= \int (2e^{2x} - e^{-2x}) dx \\ e^{2x}y &= e^{2x} + \frac{1}{2}e^{-2x} + C \\ y &= 1 + \frac{1}{2}e^{-4x} + Ce^{-2x} }$
Now using the initial condition:
$\eqalign{ y &= 1 + \frac{1}{2}e^{-4x} + Ce^{-2x} \\ 1 &= 1 + \frac{1}{2} + C \\ C &= -\frac{1}{2} }$
gives the final solution:
$\eqalign{ y = 1 + \frac{1}{2}e^{-4x} - \frac{1}{2}e^{-2x} }$
Now to determine the exact value:
$\eqalign{ y(0.5) &= 1 + \frac{1}{2}e^{-4(0.5)} - \frac{1}{2}e^{-2(0.5)} &= 0.8837 }$
and so the error is:
$\eqalign{ \Delta = \Big| \hspace{.1cm} \text{Exact} \hspace{.1cm} - \text{Approximation} \hspace{.1cm} \Big| = |0.8837 - 0.8517| = 0.0320 }$
