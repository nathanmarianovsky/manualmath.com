Problem
Given the differential equation:
$\eqalign{ \frac{dy}{dx} = xy - \frac{y}{x} }$
and the initial condition $y(1) = 1$, determine $y(1.2)$ using a $h = 0.05$ with Euler's Method up to four decimal places and compare the result to the exact solution by finding the error.
Solution
Using Euler's Method:
$n$ $x_{n + 1} = x_n + h$ $y_{n + 1} = y_n + hf(x_n,y_n)$
0 $x_1 = x_0 + h = 1.05$ $y_1 = y_0 + hf(x_0,y_0) = 1.0000$
1 $x_2 = x_1 + h = 1.10$ $y_2 = y_1 + hf(x_1,y_1) = 1.0049$
2 $x_3 = x_2 + h = 1.15$ $y_3 = y_2 + hf(x_2,y_2) = 1.0145$
3 $x_4 = x_3 + h = 1.20$ $y_4 = y_3 + hf(x_3,y_3) = 1.0287$
So according to Euler's Method $y(1.2) \approx 1.0287$. To solve for exactly separate:
$\eqalign{ \frac{dy}{dx} &= xy - \frac{y}{x} \\ \frac{dy}{y} &= (x - x^{-1}) dx \\ \int \frac{dy}{y} &= \int (x - x^{-1}) dx \\ \ln|y| &= \frac{1}{2}x^2 - \ln|x| + C \\ y &= \frac{De^{\frac{1}{2}x^2}}{|x|} }$
where:
$\eqalign{ D = e^C }$
Now using the initial condition:
$\eqalign{ y &= \frac{De^{\frac{1}{2}x^2}}{|x|} \\ 1 &= De^{\frac{1}{2}} \\ D &= e^{-\frac{1}{2}} }$
which gives the final solution:
$\eqalign{ y = \frac{e^{\frac{1}{2}x^2 - \frac{1}{2}}}{|x|} }$
Now to determine the exact value:
$\eqalign{ y(1.2) &= \frac{e^{\frac{1}{2}(1.2)^2 - \frac{1}{2}}}{|(1.2)|} &= 1.0384 }$
and so the error is:
$\eqalign{ \Delta = \Big| \hspace{.1cm} \text{Exact} \hspace{.1cm} - \text{Approximation} \hspace{.1cm} \Big| = |1.0384 - 1.0287| = 0.0097 }$
