Problem
Solve the differential equation:
$\eqalign{ \frac{dy}{dx} = \frac{3x + 2y}{3x + 2y + 2} }$
where $y(-1) = -1$.
Solution
Since the function contains a linear input make the substitution:
$\eqalign{ u &= 3x + 2y \\ u' &= 3 + 2y' }$
and plug in:
$\eqalign{ y' &= \frac{3x + 2y}{3x + 2y + 2} \\ \frac{1}{2}(u' - 3) &= \frac{u}{u + 2} \\ \frac{1}{2}u' &= \frac{u}{u + 2} + \frac{3(u + 2)}{u + 2} \\ \frac{du}{dx} &= \frac{8u + 12}{u + 2} \\ \frac{u + 2}{8u + 12} du &= dx \\ \int \frac{u + 2}{8u + 12} du &= \int dx \\ \frac{u}{8} + \frac{1}{16}\ln|2u + 3| + \frac{3}{16} &= x + C \\ \frac{3x + 2y}{8} + \frac{1}{16}\ln|6x + 4y + 3| + \frac{3}{16} - x &= C }$
Now for the constant use the initial condition:
$\eqalign{ C &= \frac{3x + 2y}{8} + \frac{1}{16}\ln|6x + 4y + 3| + \frac{3}{16} - x \\ &= \frac{3(-1) + 2(-1)}{8} + \frac{1}{16}\ln|6(-1) + 4(-1) + 3| + \frac{3}{16} - (-1) \\ &= \frac{1}{16}(9 + \ln(7)) }$
and so the final solution is:
$\eqalign{ \frac{3x + 2y}{8} + \frac{1}{16}\ln|6x + 4y + 3| + \frac{3}{16} - x = \frac{1}{16}(9 + \ln(7)) }$
