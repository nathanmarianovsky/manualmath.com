Slope Fields
In order to gain a better understanding of differential equations, slope fields provide a geometrical approach to a first order differential equation:
$\eqalign{ y' = f(x,y) }$
To view the slope field, take $\mathbb{R}^2$ and for every point evaluate the derivative. Then at each point draw a small vector and use the magnitude of the derivative to correspond to the magnitude of the vector. If the value is a large positive number, the vector will point more upwards. For values near zero the vector will point horizontally, and for very negative values it will point downwards. To make the patten clear, all of the vectors point in the "right" direction, thus all arrows will be bound between the angles $\Big[ -\frac{\pi}{2}, \frac{\pi}{2} \Big]$. $\eqalign{ y' = x }$

This happens because the derivative is independent of $y$, meaning that you can take a single horizontal section of the slope field and copy it above and below to generate the rest since they are identical. Solution Curves
With the slope field in mind, consider that the vectors drawn typically follow some sort of pattern. This patten actually helps to identify the family of solutions. Using the previous example:

The solution to this is known to be:
$\eqalign{ y = 2x - 2 + Ce^{-x} }$
and for different values of $C$ the function may behave differently. Observing closely confirms that when $C = 0$ the solution is a straight line, and for all other values the solution will behave like a line at very large values of $x$ since the exponential term becomes negligible.
