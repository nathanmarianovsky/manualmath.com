Distinct Linear Factors
Given a rational function in the form:
$\eqalign{ f(x) = \frac{P(x)}{(x-a_1)(x-a_2)\dots(x-a_n)} }$
where $P(x)$ is a polynomial of degree less than $n$ and each $a_i$ is unique, the method of partial fractions states that the function can be broken down as:
$\eqalign{ f(x) = \frac{C_1}{x-a_1} + \frac{C_2}{x-a_2} + \dots + \frac{C_n}{x-a_n} }$
where the $C$'s can be found by satisfying the equation:
$\eqalign{ P(x) &= C_1(x-a_2)(x-a_3)\dots(x-a_n) + C_2(x-a_1)(x-a_3)(x-a_4)\dots(x-a_n) \\ &+ \dots + C_n(x-a_1)(x-a_2)\dots(x-a_{n-1}) }$
Repeated Linear Factor
Given a rational function in the form:
$\eqalign{ f(x) = \frac{P(x)}{(x-a)^n} }$
where $P(x)$ is a polynomial of degree less than $n$, the method of partial fractions states that the function can be broken down as:
$\eqalign{ f(x) = \frac{C_1}{(x-a)} + \frac{C_2}{(x-a)^2} + \dots + \frac{C_n}{(x-a)^n} }$
where the $C$'s can be found by satisfying the equation:
$\eqalign{ P(x) = C_1(x-a)^{n-1} + C_2(x-a)^{n-2} + \dots + C_{n-1}(x-a) + C_n }$
Irreducible Non-Linear Factor
Given a rational function in the form:
$\eqalign{ f(x) = \frac{P(x)}{(x-b)(a_0 + a_1x + a_2x^2 + \dots a_nx^n)} }$
where $P(x)$ is a polynomial of degree less than $n+1$, the method of partial fractions states that the function can be broken down as:
$\eqalign{ f(x) = \frac{C_1}{(x-b)} + \frac{C_2 + C_3x + C_4x^2 + \dots + C_{n+1}x^{n-1}}{(a_0 + a_1x + a_2x^2 + \dots a_nx^n)} }$
where the $C$'s can be found by satisfying the equation:
$\eqalign{ P(x) = C_1(a_0 + a_1x + a_2x^2 + \dots a_nx^n) + (C_2 + C_3x + C_4x^2 + \dots + C_{n+1}x^{n-1})(x-b) }$
Repeated Irreducible Non-Linear Factor
Given a rational function in the form:
$\eqalign{ f(x) = \frac{P(x)}{(a_0 + a_1x + a_2x^2 + \dots a_nx^n)^k} }$
where $P(x)$ is a polynomial of degree less than $n+1$, the method of partial fractions states that the function can be broken down as:
$\eqalign{ f(x) &= \frac{C_{1,0} + C_{1,1}x + C_{1,2}x^2 + \dots + C_{1,n+1}x^{n-1}}{(a_0 + a_1x + a_2x^2 + \dots a_nx^n)} \\ &+ \frac{C_{2,0} + C_{2,1}x + C_{2,2}x^2 + \dots + C_{2,n+1}x^{n-1}}{(a_0 + a_1x + a_2x^2 + \dots a_nx^n)^2} \\ &+ \dots + \frac{C_{k,0} + C_{k,1}x + C_{k,2}x^2 + \dots + C_{k,n+1}x^{n-1}}{(a_0 + a_1x + a_2x^2 + \dots a_nx^n)^k} }$
where the $C$'s can be found by satisfying the equation:
$\eqalign{ P(x) &= (C_{1,0} + C_{1,1}x + C_{1,2}x^2 + \dots + C_{1,n+1}x^{n-1})(a_0 + a_1x + a_2x^2 + \dots a_nx^n)^{k-1} \\ &+ (C_{2,0} + C_{2,1}x + C_{2,2}x^2 + \dots + C_{2,n+1}x^{n-1})(a_0 + a_1x + a_2x^2 + \dots a_nx^n)^{k-2} \\ &+ \dots + (C_{k,0} + C_{k,1}x + C_{k,2}x^2 + \dots + C_{k,n+1}x^{n-1}) }$
Application Towards Integration
The method of partial fractions is a useful tool when integrating a rational function. First consider the structure of the rational function. If the degree of the numerator is the same as the degree of the denominator or greater, then perform division first:
$\eqalign{ \frac{p(x)}{q(x)} = k(x) + \frac{r(x)}{q(x)} }$
Afterwards apply the method of partial fractions on the remaining rational function on the right side.
